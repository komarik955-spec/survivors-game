# 🏚 ВЫЖИВШИЕ — Инструкция запуска

Мультиплеерная социальная игра. Аналог «Бункера» — без Unity, чисто веб.

---

## 🗂 Структура

```
survivors/
├── server/          ← Node.js + Socket.IO
│   ├── index.js     ← сервер
│   ├── gameLogic.js ← механика игры
│   └── cards.js     ← база карт
├── client/          ← React + Vite
│   └── src/
│       ├── App.jsx
│       ├── socket.js
│       └── components/
└── package.json     ← корневой (concurrently)
```

---

## 🚀 Первый запуск

### 1. Установить зависимости

```bash
cd survivors

# Корневые (concurrently)
npm install

# Сервер
cd server && npm install && cd ..

# Клиент
cd client && npm install && cd ..
```

### 2. Запустить всё одной командой

```bash
npm run dev
```

Откроется:
- **Клиент** → http://localhost:3000
- **Сервер** → http://localhost:3001

### 3. Чтобы играть на телефонах в одной сети

1. Узнай IP своего компа: `ipconfig` (Windows) или `ifconfig` (Mac/Linux)
2. В файле `client/src/socket.js` измени строку:
   ```js
   const socket = io('http://localhost:3001', ...)
   // замени localhost на твой IP, например:
   const socket = io('http://192.168.1.42:3001', ...)
   ```
3. Зайди с телефона: `http://192.168.1.42:3000`

## 🎮 Как играть

| Фаза | Что происходит |
|------|---------------|
| **Лобби** | Игроки входят, хост настраивает игру |
| **Катастрофа** | Сплэш-экран с причиной выживания |
| **Обсуждение** | Открываешь до 2 карт, убеждаешь других |
| **Голосование** | Выбираешь кого исключить (90 сек) |
| **Результат** | Показываются все карты выбывшего |
| **Финал** | Остаётся заданное количество выживших |

### Правила карт
- **6 карт** у каждого: Профессия, Здоровье, Биология, Факт, Хобби, Багаж
- **Максимум 2 карты** открывается за раунд
- Открытая карта видна **всем навсегда**

### Принудительное голосование
- Требует **50% + 1** голосов «за»
- Если все согласны — обсуждение прерывается

## 🛠 Частые проблемы

**Клиент не соединяется с сервером**
→ Убедись, что сервер запущен на порту 3001
→ Проверь `client/src/socket.js` — правильный ли адрес

**Игроки не видят друг друга**
→ Все должны быть в одной сети
→ Замени `localhost` в socket.js на IP компа

**Ошибка при npm install**
→ Нужен Node.js 18+. Проверь: `node --version`
